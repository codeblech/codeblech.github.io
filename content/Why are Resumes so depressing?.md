I feel like resumes are soon going to be outdated enough that people stop using them altogether. Why do I have to put in so much effort to fit my life's work in a page or two, in an obscure format (latex/pdf), which is almost always going to be ingested by an LLM that much better understands markdown. There are whole pipelines around processing resume PDFs so that AI can process them well. This single PDF file creates so much friction for the one writing it, and for the one reading it (LLMs).

A much better way would be to have a markdown file with as much info as one would like to be publicly available about their work. Recruiters can point their AI agents at these files and know whatever they wanna know. Of course, these agents will hallucinate; but they already do. You can't be so delusional to assume that a human would read your resume before rejecting it. It's all [thinking machines](https://dune.fandom.com/wiki/Thinking_Machines) doing the work.
